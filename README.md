# Todo List Projesi

Bu proje, basit bir Todo List uygulamasıdır. Bootstrap ve JavaScript kullanılarak geliştirilmiştir.

## Özellikler

- Todo ekleme
- Todo silme
- Tüm todoları temizleme
- Todo filtreleme

## Kullanılan Teknolojiler

- HTML5
- CSS3 (Bootstrap 4)
- JavaScript
- jQuery

## Kurulum

1. Projeyi klonlayın
2. `code.html` dosyasını bir web tarayıcısında açın
